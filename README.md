# Probability Calculator

This project is a small application that allows users to perform probability calculations.

## Technologies Used
- ASP.NET Core Web API
- React
- Axios
- xUnit (Unit Testing)

## Features
- CombinedWith calculation → P(A) * P(B)
- Either calculation → P(A) + P(B) - P(A)P(B)
- Input validation (0–1 range)
- Backend API for calculations
- Logging of calculation details
- Unit testing

## Project Structure
ProbabilityCalculatorApi → ASP.NET Core Web API
probability-calculator-ui → React frontend
ProbabilityCalculator.Tests → Unit tests

## Running the Project

### Backend

dotnet run


### Frontend

npm install
npm start


Frontend runs on:

http://localhost:3000


API runs on:

https://localhost:7219
